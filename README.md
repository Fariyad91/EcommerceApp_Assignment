# Ecommerce Mini App (React Native + TypeScript)

A simple e-commerce demo app built for assessment.  
Features product listing, product details and a functional cart with quantity control.

##  Framework Choice
**React Native (TypeScript)**  
Reason: I’m comfortable with RN architecture, components, and async APIs.  
TS ensures type safety and cleaner structure.

---

##  Features

### 1. Product Listing
- Fetches products from FakeStore API  
- Shows loading + error states  
- Displays image, title, price and category

### 2. Product Detail
- Shows full product information  
- Add to cart button  
- Rating, description etc.

### 3. Shopping Cart
- Increase/decrease quantity  
- Remove item  
- Auto calculates totals  
- Checkout button (dummy confirmation)

### 4. Navigation
- Bottom Tabs + Stack  
- Badge on cart icon  
- Splash screen included

---

##  State Management
- **Context API**  
Used for cart state across screens.
